
SPA 8/31版（PWA配布パッケージ）

1) この ZIP を解凍して、フォルダ内のファイル（index.html / manifest.webmanifest / sw.js / icons/）を
   GitHub リポジトリのルートにドラッグ&ドロップでアップロード → Commit changes
2) Settings → Pages → Source: Deploy from a branch / Branch: main / Folder: /(root) → Save
3) 公開URLをスマホで開き、ホーム画面に追加（iPhone: Safariの共有 / Android: Chromeのメニュー）

※ iOS の音声/GPSは必ず HTTPS 上で。
※ 既にホーム画面に古い版がある場合は、削除→URLを開き直して追加（キャッシュ更新）。
